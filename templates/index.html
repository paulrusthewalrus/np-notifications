Neptune's Pride Notification System

by Paul!

You're only seeing the front-end side of a rather extensive back-end system. At the moment, we have nothing here.

These are some of the technologies being used at the moment to get this off the ground:

  • Firebase/Firestore (Database for storing fleet information / message notifications etc.)
  • Twilio API (Sending texts to your phones)
  • The Neptune's Pride API (Making requests to the game and getting relevant information)
  • Google Cloud App Engine (Hosting this app)
  • Google Cloud Scheduler (Constantly makes requests to the back-end to check for updates)

The back-end here is built in Python, which parses the game data, interacts with the database, and makes third party calls to APIs.
